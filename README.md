# WeAppening
by mercury

*WeBeAppening* is a web/mobile app to connect people who want to do an activity with others who would like to tag along.

Features
1. Profile
2. Timeline
3. Social Media Share
4. Comment
5. Photo Upload
6. Chat(private)
